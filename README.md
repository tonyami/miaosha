# miaosha

单机版移至 v1 分支

[在线演示](https://mtony.cn/miaosha/)

## 更新计划

- [ ] 基于 etcd+grpc+protobuf 实现微服务架构
- [ ] 基于 rabbitmq 实现异步下单、订单超时关闭
- [ ] 基于支付宝当面付实现扫码支付功能

- [ ] 基于微信小程序实现扫码登录功能

- [ ] 新增后台管理系统
- [ ] 新增 PC 端入口
